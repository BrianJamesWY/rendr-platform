# Rendr - Multi-Source Video Verification Platform

Solving the $78 billion truth crisis through advanced multi-source video verification technology.

## Features

- **Multi-Source Verification**: Compare multiple recordings of the same event to prove authenticity
- **Advanced Audio Fingerprinting**: Spectral analysis and peak matching algorithms
- **Computer Vision Analysis**: SIFT feature detection and visual correlation
- **GPS & Timestamp Verification**: Spatial-temporal correlation analysis
- **Device Metadata Analysis**: Authenticity verification through device fingerprinting
- **Real-time Processing**: Fast verification with detailed reporting

## Technology Stack

- React 18
- Tailwind CSS
- Lucide React Icons
- Advanced algorithms for audio/visual processing

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```
